Most commonaly used commands:

querySelector() - returns the first instance of all matching elements
Syntax example: let ele = document.querySelector('div');

querySelectorAll() - returns all elements that match the specified CSS selector
Syntax example: let ele = document.querySelectorAll('p');


getElementById() - targets a particular Id by name in HTML doc
Syntax example: let id = document.getElementById('id')


getElementsByClassName() - targets particular classes by name in HTML doc
Syntax example: let eles = document.getElementsByClassName('div')

getElementsByTagName() - targets elements given a tag name in a HTML doc
Syntax example: let tags = document.getElementsByTagName(tagname)


createElement() - creates an element in a HTML doc
Syntax example: const heading = document.createElement("h1)

createTextNode() - creates a text node in a HTML doc
Syntax example: const heading_text = document.createTextNode("This Heading!")
